A ouvidoria é um sistema que conta com dois agentes o ouvidor e o cliente, para isso o sistema foi dividido em duas partes, uma parte WEB, e outra em forma de aplicativo mobile, a aplicação web foi elaborada para atender a parte corporativa da ouvidoria, que são as empresas de ônibus responsáveis por captar as mensagens dos usuários, já para os clientes (usuários de ônibus) foi desenvolvido um aplicativo, que vai servir para que eles possam mandar mensagens para as empresas de ouvidoria, e assim possibilitando uma comunicação fluida e efetiva em ambas as partes.
